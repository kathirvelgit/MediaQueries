# 📱 Responsive Web Layout with CSS Media Queries

A simple responsive web layout built using **HTML5** and **CSS3** to demonstrate the power of **CSS Flexbox** and **Media Queries**. This project showcases how webpage layouts and text sizes adapt to different screen sizes, providing a better user experience across desktops, tablets, and mobile devices.

## 📈 Responsive Breakpoints

| Screen Width        | Layout Behavior                            |
| ------------------- | ------------------------------------------ |
| Greater than 1746px | Horizontal Flex Layout                     |
| 1746px or less      | Vertical Layout (`flex-direction: column`) |
| 603px or less       | Font size changes to **12px**              |
| 349px or less       | Font size changes to **10px**              |
